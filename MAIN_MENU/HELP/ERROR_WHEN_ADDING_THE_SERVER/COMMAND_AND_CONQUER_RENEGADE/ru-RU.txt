Серверы с игрой "Command and Conquer: Renegade" кроме порта для подключения используют "GameSpy(Query) порт".

Поэтому для добавления необходимо ввести не только IP  и порт, а так же "GameSpy(Query) порт" сервера.

Который можно узнать, связавшись с владельцами серверов, которые вы хотите добавить в мониторинг. Если вы владелец такого сервера, то информацию нужно искать в конфигурации вашего сервера.

Стандартный Query-порт: 25300, он используется на большинстве серверов.
