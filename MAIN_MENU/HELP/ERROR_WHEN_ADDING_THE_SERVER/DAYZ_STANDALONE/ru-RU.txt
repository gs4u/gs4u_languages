Если не получается добавить ваш сервер по основному порту, то вам необходимо найти в настройках сервера какой Query порт использует ваш сервер и указать его вторым "Query" портом при добавлении.

Для того чтобы найти Query порт для вашего сервера, вам нужно найти файл "Start_server_parameters.ini" в данном файле в разделе ищите раздел:
[Networking]
тут будет указан "ListenPort"

Пример:

...
[Networking]
BroadcastAddress = ...
ListenPort = 12345
...
Этот порт и есть Query порт. Его нужно указать вторым при добавлении в мониторинг.
